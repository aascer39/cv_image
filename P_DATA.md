## 2. 纵向技术演进脉络与技术节点详情 (网站时间轴与方法页核心素材)

### 📊 第一阶段：传统手工特征与经典模式识别时代 (1999 - 2011)

本阶段的核心逻辑是“人工根据先验知识设计特征算子” + “传统分类器进行浅层模式识别”。

- **SIFT (Scale-Invariant Feature Transform, 1999)**：
  - _技术细节_：尺度不变特征变换。通过在多尺度高斯差分空间（DoG）中检测极值点，并利用局部梯度方向直方图确立主方向。
  - _核心优势_：对图像的旋转、尺度缩放、亮度变化保持极强的鲁棒性，对视角变化、仿射变换和噪声也具备一定稳定性。
  - _应用场景_：早期图像拼接、物体识别、三维重建。
- **HOG (Histogram of Oriented Gradients, 2005)**：
  - _技术细节_：方向梯度直方图。通过计算和统计图像局部区域的梯度方向直方图来构成特征。
  - _核心优势_：由于在局部细胞单元（Cells）和块（Blocks）上进行了梯度归一化，因此对光照变化和阴影具备很强的容忍度，能够完美表征物体的边缘和形状轮廓。
  - _应用场景_：静态图像和视频中的行人检测（经典组合：HOG + SVM）。
- **LBP (Local Binary Pattern, 2002)**：
  - _技术细节_：局部二值模式。一种用来纹理描述的算子。将中心像素与其 3x3 邻域的灰度值进行阈值比较，大于等于为 1，小于为 0，从而产生 8 位二进制数表征局部纹理。
  - _核心优势_：计算极其简单、具备极佳的旋转不变性和灰度不变性。
  - _应用场景_：人脸识别、表面纹理缺陷检测。
- **经典分类器基础**：
  - 支持向量机 (SVM)：寻找一个最优超平面实现最大间隔分类。
  - 随机森林 (Random Forest)：集成多个决策树进行投票。
  - AdaBoost：通过迭代改变样本权重，将多个弱分类器组合成强分类器。

### 🚀 第二阶段：深度卷积神经网络（CNN）夺冠与统治时代 (2012 - 2019)

本阶段的核心逻辑是“端到端深度特征自适应学习”，特征提取由网络在训练中自行优化。

- **AlexNet (2012)**：
  - _历史意义_：夺得 2012 年 ImageNet 挑战赛冠军，拉开深度学习统治计算机视觉的序幕。
  - _创新特性_：首次采用 GPU 进行大规模并行加速计算；引入 ReLU 激活函数解决深层网络中的梯度饱和问题；使用 Dropout 抑制过拟合；引入数据增强技术。
- **VGGNet (2014)**：
  - _核心贡献_：系统研究了“网络深度”对模型性能的影响。
  - _技术特征_：废弃了早期的大型卷积核，全站通过堆叠 3x3 的小卷积核和 2x2 的最大池化层（Max Pooling）来构建 16-19 层的深层网络。证明了多个小卷积核叠加具有与大卷积核相同的感受野，且能引入更多非线性表达、减少参数量。
- **GoogLeNet (Inception, 2014)**：
  - _核心创新_：提出 Inception 模块，在“网络宽度”上进行横向扩张。在同一层并行执行 1x1, 3x3, 5x5 卷积以及 3x3 最大池化，最后将特征图拼接，大幅提升了对多尺度特征的自适应感知能力，并利用 1x1 卷积实现了完美的降维与参数压缩。
- **ResNet (残差网络, 2015)**：
  - _划时代突破_：何恺明提出的残差连接（Skip Connection / Shortcut）彻底改写了深度学习历史。
  - _解决痛点_：网络层数过深时（如超过50层），会导致网络产生“退化问题”（Degradation Problem），即训练集和测试集准确率同时下降。残差连接让前一层的信息可以跨层直接流动，使得网络可以平滑拓展至 152 层甚至上千层。

### 🎯 第三阶段：目标检测与图像分割的双轨跃进

将分类能力拓展到密集像素、空间边界的精细感知。

- **目标检测两阶段（Two-stage）流派**：
  - _R-CNN / Fast R-CNN_：将区域建议（Region Proposals）与 CNN 特征提取结合，逐步解决了候选区域特征重复计算的资源浪费。
  - _Faster R-CNN_：引入了区域生成网络（RPN），将候选框提取、特征提取、分类与边界框回归完全集成到一个端到端的深层网络中，大幅提升了检测速度和精度。
  - _Mask R-CNN_：在 Faster R-CNN 基础上增加了一个平行的 ROI Align 和像素级掩码（Mask）生成分支，实现了完美的目标检测与实例分割统一。
- **目标检测单阶段（One-stage）流派**：
  - _YOLO 系列 (You Only Look Once)_：将检测问题直接视作一个单一的端到端回归问题，直接在输出层输出边界框坐标与类别概率，具备工业级、极高帧率的实时检测能力。
  - _SSD (Single Shot MultiBox Detector)_：利用多个不同尺度的特征图分别进行检测，有效改善了单阶段算法对小目标检测能力不足的弊端。
- **图像分割技术演进**：
  - _FCN (Fully Convolutional Networks)_：全卷积网络，首次将传统 CNN 的全连接层替换为全卷积层，配合上采样/反卷积，率先实现像素级、端到端的目标分割。
  - _U-Net (2015)_：专为医学影像设计的完美对称 U 型架构。通过左侧编码器（下采样）提取语义上下文，右侧解码器（上采样）恢复空间分辨率，中间通过跳跃连接（Skip Connection）进行特征拼接，完美保留了医学图像中极其宝贵的边缘边界信息。
  - _DeepLab 系列_：引入空洞卷积（Atrous Convolution）在不增加参数量的前提下成倍扩大感受野，并配合空间金字塔池化（ASPP）捕捉多尺度上下文。

### 🌐 第四阶段：Transformer 大模型与 AIGC 时代 (2020 - 2026 当前)

打破局部感受野限制，走向全局自注意力感知与开放域大模型泛化。

- **Vision Transformer (ViT, 2020)**：
  - _技术质变_：证明了标准的 Transformer 架构也可以直接应用到计算机视觉任务中。它将一幅图像切分成一系列固定的 Patch（如 16x16 像素块），线性投影后加上位置编码（Position Embedding），直接投入经典的 Self-Attention 机制。在大规模数据集预训练下，彻底超越了 CNN 的性能上限。
- **Swin Transformer**：
  - _优化改进_：提出基于移动窗口（Shifted Windows）的局部自注意力机制，将计算复杂度从图像大小的平方阶降低到了线性阶，兼顾了全局感知力与高效的计算开销。
- **SAM (Segment Anything Model, 2023)**：
  - _历史地位_：计算机视觉领域的“ChatGPT 时刻”，Meta 发布的视觉分割大模型。
  - _核心能力_：基于 11 亿张掩码的庞大数据集预训练，支持通过点（Points）、框（Boxes）、文本（Texts）等多种交互式提示进行零样本（Zero-shot）通用目标分割。
- **生成式模型与 AIGC**：
  - _GAN (Generative Adversarial Networks)_：通过生成器与判别器的博弈对抗提升图像保真度。
  - _Diffusion Models (扩散模型 / Stable Diffusion)_：通过前向加噪与反向去噪过程建模，成为当前 AIGC 图像生成的绝对底层基石。

## 4. 落地应用场景 (约会/应用页的高亮内容)

- **智慧安防**：人脸识别定位、人流密度监控、异常行为实时检测（核心技术：YOLOv8/v10 实时检测）。
- **自动驾驶**：车载摄像头环视感知、多视角车道线检测、BEV（鸟瞰图）时空融合感知（核心技术：Transformer + 语义分割）。
- **医学影像分析**：CT/核磁共振图像的病灶区全自动精准提取、皮肤癌结节像素级分割（核心技术：改进版 3D U-Net）。
- **遥感测绘**：高分辨率卫星遥感图像的地物分类、建筑物群和植被覆盖率自动测算（核心技术：遥感专有场景语义分割）。
- **工业缺陷检测**：流水线产品表面微小划痕、裂纹、杂质的工业级极高速机器视觉无损检测（核心技术：异常检测+超分辨率 SwinIR 算法）。
